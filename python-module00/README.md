# Python Module 00 - Growing Code

## Description

Growing Code is an introduction to Python fundamentals through community garden scenarios. The project covers basic concepts like variables, functions, control flow, loops, and type annotations.

## Exercises

| Exercise | File | Description |
|----------|------|-------------|
| ex0 | ft_hello_garden.py | Display a welcome message |
| ex1 | ft_plot_area.py | Calculate garden plot area |
| ex2 | ft_harvest_total.py | Sum harvest weights over 3 days |
| ex3 | ft_plant_age.py | Check if plant is ready to harvest |
| ex4 | ft_water_reminder.py | Remind to water plants |
| ex5 | ft_count_harvest_iterative.py | Count days (loop) |
| ex5 | ft_count_harvest_recursive.py | Count days (recursion) |
| ex6 | ft_garden_summary.py | Display garden info |
| ex7 | ft_seed_inventory.py | Seed inventory with type hints |
